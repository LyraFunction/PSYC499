# Honors in Psychology Thesis
## Presidential Election Vote Outcome as a Predictor in County Level Sexual Orientation and Gender Motivated Hate Crimes across the United States

## Overview
This repo contains all the materials for my undergraduate thesis.

## Repository Structure
-   `data/raw_data` will contain the pertinent files for the project along with ris files for citations for the data.
-   `data/clean_data` will contain the final files clean in Python for use in the R scripts.
-   `scripts` will contain the R, LaTeX, and Python files used for the project along with any relevent documention.
-   `paper` will contain the PDFs submitted for the project, including the proposal, defense, and final drafts.
-   `supplementals` will contain supplemental information such as rational for data decisions.
